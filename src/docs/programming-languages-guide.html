🖥️ How to Make .EXE Files

💡 Key Insight
You DON'T need to learn a new language for .exe! Tauri wraps your existing JS/HTML/CSS into a tiny Windows executable. Same code, two outputs: web + desktop.
⚡Tauri 🌸 RECOMMENDED▼
Your JS app → ~3MB .exe. Uses system webview (not bundled Chrome). Backend in Rust (you don't need to learn Rust!).
.exe size
~3MB
Speed
⚡
🔗 tauri.app • License: MIT (free)
⚙️Electron▼
Same idea but bundles entire Chrome browser. VS Code, Discord, Slack use it. Heavy but battle-tested.
.exe size
~150MB
Speed
Medium
🔗 electronjs.org • License: MIT (free)
🐍Python + PyInstaller▼
Converts Python scripts to .exe. Good for tools/scripts, not great for GUI apps.
.exe size
~50-200MB
🔗 pyinstaller.org • License: GPL (free)

Microsoft's native app framework. Great for Windows-first apps. Visual Studio required.

☁️ Deployment — Where Does Your App Live?

How to get your app from your computer → the internet (or a .exe on someone's desktop)
🌸 What IS Deployment?
Deployment = putting your files somewhere people can access them.
Your app lives on your computer now. Deployment copies it to a server in the cloud so anyone with the URL can use it. ?
🔀Decision Map — Which Deployment?▼
📦Deployment Formats — What Gets Uploaded?▼
📄
Static Files
HTML + CSS + JS files uploaded as-is.
No server processing.
This is YOU! 🌸
⚙️
Serverless Functions
Small code that runs on demand.
AWS Lambda / Azure Functions.
Pay per execution, not per hour.
🐳
Containers (Docker)
Your app + all dependencies packaged.
Runs anywhere identically.
For complex backends.
🖥️
.EXE / Installer
Desktop distribution.
Tauri / Electron → .exe or .msi
Distributed via website or store.

🚀 How to Deploy YOUR App to AWS:
# 1. Create S3 bucket aws s3 mb s3://my-feature-analyser # 2. Upload files aws s3 sync ./features_analyser/ s3://my-feature-analyser/ --delete # 3. Create CloudFront distribution (points to S3) # → Console: CloudFront → Create Distribution → Origin: S3 bucket # 4. Get HTTPS certificate (free!) # → Console: ACM → Request Certificate → your-domain.com # 5. Done! Your app is live at: # https://d1234abc.cloudfront.net 🌸
