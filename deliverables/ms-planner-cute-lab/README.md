# Microsoft Planner Deep Map + Cute Gap Lab

This package is a standalone interactive HTML artifact.

What is inside:

- `index.html`
- `css/main.css`
- `js/data.js`
- `js/renderers.js`
- `js/app.js`

What it does:

- maps current Microsoft Planner features
- separates Basic vs Premium vs Plan 3 / Plan 5
- summarizes current complaint themes from official + community + review sources
- identifies true missing features vs tier-gated features
- includes 40 functional module mockups
  - 30 existing-feature extension modules
  - 10 missing-feature build modules
